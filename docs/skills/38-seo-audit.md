# SEO Audit Skill (User-Provided)

## What It Does
Performs comprehensive SEO audits covering technical SEO, on-page optimization, meta tags, Core Web Vitals, crawl errors, and indexing issues. Use when diagnosing why a site isn't ranking or traffic has dropped.

## Activation Triggers
- "SEO audit", "why am I not ranking", "SEO issues"
- "technical SEO", "meta tags review", "SEO health check"
- "my traffic dropped", "lost rankings", "not showing up in Google"
- "page speed", "core web vitals", "crawl errors", "indexing issues"

## Audit Categories

### Technical SEO
- Crawlability (robots.txt, sitemap.xml)
- Page speed (Lighthouse scores, LCP, FID, CLS)
- Mobile-friendliness
- HTTPS security
- Canonical tags
- Structured data (JSON-LD)
- Hreflang for international

### On-Page SEO
- Title tag optimization (50-60 chars)
- Meta description (150-160 chars)
- H1/H2/H3 hierarchy
- Keyword density and placement
- Image alt text
- Internal linking structure

### Content SEO
- Keyword research and targeting
- Content depth and completeness
- E-E-A-T signals (Experience, Expertise, Authority, Trust)
- Featured snippet optimization
- FAQ schema for voice search
