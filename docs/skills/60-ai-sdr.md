# Skill: ai-sdr

## Purpose
Functions as an AI Sales Development Representative (SDR) — generates cold outreach sequences, personalized emails, LinkedIn messages, follow-up cadences, objection handlers, and sales scripts for B2B prospecting.

## When to Use
- Need cold email sequences for outbound sales
- Need LinkedIn connection request + follow-up messages
- Need objection handling scripts
- Need discovery call scripts
- Need a full outreach cadence (multi-touch, multi-channel)

## Core Deliverables
| Output | Description |
|--------|-------------|
| Cold Email Sequence | 5-7 email cadence with subject lines |
| LinkedIn Messages | Connection request + 3 follow-up messages |
| Cold Call Script | Opener + qualification + pitch + CTA |
| Objection Handlers | 10-15 common objections with responses |
| Follow-up Cadence | Day 1/3/7/14/21 touchpoint plan |
| ICP Definition | Ideal Customer Profile with firmographics |
| Personalization Snippets | 5 openers based on prospect trigger events |

## Usage Examples

```
"Write a 5-email cold outreach sequence for selling ISHU TOOLS Pro to small business owners"
"Generate LinkedIn messages for connecting with CTOs at Series A startups in India"
"Create objection handlers for 'We already use Google Workspace for everything'"
"Write a cold call script for selling an AI writing tool to content marketing agencies"
```

## Cold Email Framework (AIDA)

### Email 1 — Attention (Day 1)
```
Subject: [Trigger/Relevance] — quick thought

Hi [Name],

[Personalized observation about their company/role].

[Problem you solve in 1 sentence].

[Social proof or specific result].

Worth a 15-min call this week?

[Name]
```

### Email 2 — Interest (Day 3)
```
Subject: Re: [previous subject]

[Name], thought this case study might be relevant...
[Brief story: similar company + result achieved]
```

### Email 3 — Value (Day 7)
```
Subject: [Specific benefit] for [Company name]

[Lead with specific value proposition]
[ROI or time savings metric]
```

### Email 4 — Urgency (Day 14)
```
Subject: Last check-in

[Low-pressure close with easy next step]
```

### Email 5 — Breakup (Day 21)
```
Subject: Closing the loop

[Permission to close + leave door open]
```

## Personalization Triggers
- **Funding news**: "Congrats on the Series B — scaling teams often means..."
- **New job**: "Just saw you started at [Company] — typically the first 90 days..."
- **Product launch**: "Noticed you launched [feature] — teams expanding like this often..."
- **Hiring signal**: "Saw you're hiring 5 engineers — companies at this stage often need..."
- **Content**: "Your recent post on [topic] resonated — we've been thinking the same..."

## Objection Handler Matrix
| Objection | Response Framework |
|-----------|-------------------|
| "Not interested" | Curiosity close: "Totally fair — can I ask what [problem] looks like for you today?" |
| "Too expensive" | ROI reframe: "What's the cost of [pain] per month?" |
| "We already use X" | Differentiation: "We work alongside X — here's what's different..." |
| "Send me an email" | Confirm intent: "Of course. To make it relevant, quick question first..." |
| "Not the right time" | Future anchor: "When would be the right time? Can I follow up in 30 days?" |

## ICP Definition Template
```
Company: B2B SaaS, 20-200 employees, Series A-B
Revenue: $1M-$20M ARR
Geography: India, SEA, US
Department: Marketing, Operations, Product
Title: VP, Director, Head of, Manager
Pain: [Specific pain this product solves]
Trigger: Hiring spike, new funding, product launch
```
