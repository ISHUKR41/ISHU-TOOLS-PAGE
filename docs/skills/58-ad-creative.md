# Skill: ad-creative

## Purpose
Generate high-converting ad creatives — copy, headlines, CTAs, and visual direction — for Google Ads, Facebook/Instagram Ads, LinkedIn Ads, YouTube pre-rolls, and display networks. Produces multiple variants for A/B testing.

## When to Use
- User needs ad copy for paid campaigns
- User wants multiple headline/description variants for Google RSAs
- User needs Facebook ad body copy + primary text
- User wants YouTube ad scripts (5-sec skippable or 15-sec non-skip)
- User wants banner ad copy with strong CTAs

## Key Capabilities
| Output | Format |
|--------|--------|
| Google Ads RSA | 15 headlines (≤30 chars) + 4 descriptions (≤90 chars) |
| Facebook/Instagram | Primary text + headline + description + CTA button |
| LinkedIn Ads | Sponsored content copy + InMail templates |
| YouTube | Hook (0–5s) + body (5–15s) + CTA (15–30s) scripts |
| Display/Banner | Tagline + body + CTA (short, punchy) |

## Usage Pattern

```
"Create 3 Facebook ad variants for my SaaS tool targeting freelancers aged 25-35 in India"
"Write Google RSA headlines and descriptions for my PDF converter tool"
"Generate YouTube 30-second ad script for ISHU TOOLS targeting students"
```

## Ad Frameworks Used
- **AIDA**: Attention → Interest → Desire → Action
- **PAS**: Problem → Agitate → Solution
- **4Cs**: Clear, Concise, Compelling, Credible
- **BAB**: Before → After → Bridge
- **Hook Model**: Trigger → Action → Variable Reward → Investment

## Parameters to Specify
- **Product/Service**: What you're advertising
- **Target Audience**: Demographics, interests, pain points
- **Goal**: Awareness / Traffic / Conversions / App installs
- **Platform**: Google / Facebook / LinkedIn / YouTube / Display
- **Tone**: Professional / Playful / Urgent / Inspirational
- **Budget Tier**: Budget ($10/day) / Mid ($100/day) / Scale ($1000/day)
- **USP**: Key differentiator over competitors

## Output Quality Checklist
✅ Emotional trigger in headline  
✅ Benefit-focused (not feature-focused)  
✅ Clear CTA with urgency or value  
✅ Character limits respected per platform  
✅ At least 3 A/B variants provided  
✅ Mobile-first copy (short sentences)
