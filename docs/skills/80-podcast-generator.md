# Skill: podcast-generator

## Purpose
Creates all aspects of a podcast — show concept, episode outlines, full scripts, intro/outro text, guest interview questions, show notes, chapter markers, and RSS feed structure. Also helps with podcast production planning.

## When to Use
- Launching a new podcast and need show format/concept
- Need episode scripts for educational or interview-style shows
- Need show notes for SEO (podcast description + keywords)
- Need guest interview question banks
- Need episode titles and descriptions that attract listeners
- Need a podcast production checklist

## Podcast Types Supported
| Format | Structure | Best For |
|--------|-----------|---------|
| Solo | One host, no guest | Education, commentary, tips |
| Interview | Host + guest | Expertise sharing, stories |
| Co-hosted | 2-3 hosts | Conversational, debate |
| Storytelling | Narrative | True crime, fiction, history |
| Panel | Multiple guests | Industry roundtables |
| Micro | < 10 minutes | Daily news, quick tips |

## Usage Examples

```
"Create a podcast concept for a show about Indian student life at IITs"
"Write a 30-minute interview script with questions for a guest who's a startup founder"
"Generate episode titles and descriptions for a tech tools podcast"
"Write show notes with keywords for episode about 'Free PDF Tools for Students'"
"Create a Q&A script for interviewing Ishu Kumar about building ISHU TOOLS"
```

## Podcast Show Bible Template

### Show Concept
```
SHOW TITLE: "Tools & Tricks" by ISHU TOOLS
TAGLINE: "Free tools, smart hacks, zero cost"
HOST: Ishu Kumar (IIT Patna)
FORMAT: Solo tips + occasional guest interviews
DURATION: 15-25 minutes per episode
FREQUENCY: Weekly (every Monday)
TARGET AUDIENCE: Indian students and young professionals
LANGUAGE: Hinglish (Hindi + English mix)

UNIQUE VALUE PROPOSITION:
"Every episode teaches one powerful free tool or study hack that 
saves you time or money. No fluff, no sponsorships, just value."

SHOW STRUCTURE:
[0:00] Intro music + hook (30 sec)
[0:30] Episode overview (1 min)
[1:30] Main content (15-20 min)
[18:00] Quick recap (2 min)
[20:00] Listener tip of the week (1 min)
[21:00] Outro + CTA (1 min)
```

### Episode Outline Template
```
EPISODE #[Number]: "[Catchy Title]"
Published: [Date]
Duration: [Target length]

HOOK (30 sec):
"[Surprising fact or question that creates curiosity]"

INTRO (1 min):
"Today we're talking about [topic]. By the end of this episode,
you'll know [specific outcome]. Let's dive in."

SEGMENT 1: [Topic/Problem] (5 min)
- What's the problem?
- Who faces this? (relatable scenario)
- How most people solve it (inefficiently)

SEGMENT 2: [Solution/Tool/Technique] (10 min)
- What is the solution?
- Step-by-step how-to
- Pro tips and tricks
- Common mistakes to avoid

SEGMENT 3: Examples/Case Studies (5 min)
- Real example with numbers/results
- Listener success story or anecdote

OUTRO (1 min):
"Today we covered [X]. Next week, I'll talk about [Y].
Subscribe so you don't miss it. Links are in the description.
I'm [Name], and this is [Show name]. Until next week!"
```
