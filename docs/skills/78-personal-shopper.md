# Skill: personal-shopper

## Purpose
Acts as an AI personal shopping assistant — recommends products based on budget, preferences, and use case; compares products across categories; finds best deals; and provides buying guides for electronics, fashion, home goods, and more in the Indian market context.

## When to Use
- User needs laptop/phone recommendations within a budget
- User wants to compare two or more specific products
- User needs a buying guide for a product category
- User wants to know what to look for when buying something
- User needs to find best value products for a specific use case
- User wants gift ideas

## Buying Guide Framework

### For Any Product
```
Step 1: IDENTIFY NEEDS
  - Primary use case
  - Secondary features
  - Deal-breakers (must-have vs nice-to-have)

Step 2: SET BUDGET
  - Total budget
  - "Value zone" (avoid bottom 20% and top 20% of market)

Step 3: RESEARCH SPECS
  - Category-specific key specs (see below)
  - Reviews: YouTube reviews > Amazon reviews

Step 4: SHORTLIST 3-5 OPTIONS
  - Best value, mid-range, premium pick

Step 5: CHECK PRICE HISTORY
  - Camelcamelcamel (Amazon), Smartprix (India)
  - Best prices: Flipkart during Big Billion Days, Amazon Great Indian Festival

Step 6: BUY & RETURN POLICY
  - 10-day return window on Flipkart
  - Amazon typically 30 days for electronics
```

## Key Specs Cheat Sheet

### Laptop Specs to Look For
```
MUST CHECK:
✅ Processor: Ryzen 5 ≥ Intel i5 (for value)
✅ RAM: 8GB minimum, 16GB better
✅ Storage: SSD (not HDD) — 512GB minimum
✅ Display: FHD (1920×1080) minimum, IPS panel
✅ Battery: Watt-hours (Wh) > advertised hours

NICE TO HAVE:
✓ Backlit keyboard
✓ USB-C charging
✓ USB 3.0+ ports
✓ Fingerprint reader
✓ Dual-band WiFi (802.11ax/WiFi 6)

AVOID:
❌ HDD (mechanical hard disk)
❌ 4GB RAM in 2026
❌ HD (1366×768) display
❌ Celeron/Pentium processors
```

### Earphone/Headphone Specs
```
Key Specs:
• ANC (Active Noise Cancellation): Premium feature
• Battery: 6h earbuds + 18h case = good
• Latency: < 50ms for gaming, < 100ms acceptable
• Driver size: 10-12mm for earbuds (bigger = better bass)
• Codec: aptX/LDAC > SBC (audio quality over Bluetooth)
• IP rating: IPX4 = sweat resistant (gym use)
```
