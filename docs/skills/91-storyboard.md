# Skill: storyboard

## Purpose
Creates visual storyboard scripts — scene-by-scene descriptions, camera angle notes, dialogue, action descriptions, and shot lists — for videos, animations, films, commercials, explainer videos, and social media content.

## When to Use
- Creating a storyboard for a product demo video
- Planning a YouTube tutorial with multiple scenes
- Designing an animated explainer video
- Creating a commercial/ad video storyboard
- Planning a social media video series
- Creating a presentation narrative structure

## Storyboard Format

### Shot-by-Shot Template
```
STORYBOARD: [Project Title]
Duration: [Total expected length]
Format: [16:9 horizontal / 9:16 vertical / 1:1 square]
Style: [Live action / Animation / Screen recording / Mixed]

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SCENE 1 | Duration: 0:00 - 0:05
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VISUAL:
┌─────────────────────────────────────┐
│                                     │
│   [Scene description: what we SEE]  │
│   Camera: Close-up / Wide / Medium  │
│   Angle: Eye level / Above / Below  │
│                                     │
└─────────────────────────────────────┘

AUDIO:
Voice Over: "[Narration text]"
Music: [Background music type/mood]
SFX: [Sound effects if any]

ACTION:
[What happens in this frame — movement, transitions]

TEXT OVERLAY:
"[Any text shown on screen]"

TRANSITION to Scene 2: Cut / Fade / Wipe / Zoom
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Shot Types Reference
```
SHOT TYPES:
ELS — Extreme Long Shot (landscape, setting)
LS  — Long Shot (full body/product)
MS  — Medium Shot (waist up or product mid-view)
CU  — Close-Up (face, detail, specific element)
ECU — Extreme Close-Up (eye, button, single detail)

CAMERA MOVEMENTS:
Pan  — Horizontal camera movement
Tilt — Vertical camera movement
Zoom — Magnification change
Dolly — Camera physically moves forward/back
Tracking — Camera follows subject

TRANSITIONS:
Cut — Instant scene change
Fade — Gradual fade to black then new scene
Dissolve — Blend between two scenes
Wipe — New scene "wipes" over old
Match Cut — Same framing, different content
```
