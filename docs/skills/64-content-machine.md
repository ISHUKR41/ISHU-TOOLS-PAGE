# Skill: content-machine

## Purpose
Creates systematic, high-volume content strategies and outputs — blog posts, social media calendars, video scripts, newsletters, pillar pages, and content clusters — designed to drive organic traffic and audience growth at scale.

## When to Use
- Need a 30/90-day content calendar
- Need blog post outlines + full drafts
- Need LinkedIn/Twitter/Instagram post series
- Need a YouTube video script (educational or promotional)
- Need a newsletter issue (weekly/monthly)
- Need a content cluster strategy for SEO

## Core Capabilities
| Content Type | Volume Possible |
|-------------|-----------------|
| Blog Posts | Full 1500-3000 word drafts |
| Social Posts | 30-day calendar in one shot |
| Video Scripts | 5-30 minute scripts with B-roll notes |
| Newsletter | Full issue with subject lines, sections, CTA |
| Pillar Pages | 5000+ word comprehensive guides |
| Content Clusters | Hub + 10-15 spoke articles |
| Thread Content | Twitter/X and LinkedIn thread series |
| YouTube Shorts Scripts | 30-60 second hook + value + CTA |

## Usage Examples

```
"Create a 30-day content calendar for ISHU TOOLS targeting students on Instagram and LinkedIn"
```

## Blog Post Structure (SEO-Optimized)
```
TITLE: [Primary keyword + benefit + year]
META: [160 chars with keyword + CTA]

H1: [Title repeated or variation]
  INTRO (150 words):
  - Hook: stat/question/story
  - Problem statement
  - Promise: what reader will learn
  - [Target keyword used naturally]

H2: [Subheading 1 with keyword variation]
  Body (200-300 words)
  [Bullet points or numbered list]

H2: [Subheading 2]
  ...

H2: FAQs (5-7 questions — targets featured snippets)
  Q: [Common question]
  A: [Direct 40-60 word answer]

CONCLUSION:
  - Summary
  - CTA: Try the tool / share the post / subscribe

INTERNAL LINKS: 3-5 links to related tools/posts
EXTERNAL LINKS: 1-2 authoritative sources
IMAGES: Suggested alt text for each image
```

## Content Calendar Template (30-Day)
```
Week 1 — AWARENESS (Reach new audience)
Mon: [Educational post — problem you solve]
Wed: [How-to tutorial — your tool in action]
Fri: [User success story or stat]

Week 2 — CONSIDERATION (Nurture interest)
Mon: [Comparison post — you vs alternative]
Wed: [Deep dive — one specific feature/tool]
Fri: [FAQ roundup from common questions]

Week 3 — CONVERSION (Drive action)
Mon: [Case study — before/after using tool]
Wed: [Step-by-step guide with screenshots]
Fri: [Limited offer or feature announcement]

Week 4 — RETENTION (Keep existing users)
Mon: [Tips and tricks power-users should know]
Wed: [Behind the scenes / creator story]
Fri: [Community spotlight / user submission]
```
