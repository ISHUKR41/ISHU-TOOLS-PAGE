# Skill: branding-generator

## Purpose
Creates comprehensive brand identity systems — brand name, tagline, color palette, typography, tone of voice, logo concept, brand guidelines, and visual language — for startups, products, and personal brands.

## When to Use
- Starting a new product or company and need brand identity
- Rebranding an existing product
- Need a color palette and typography system
- Need a brand voice/tone guidelines document
- Need a brand name + domain availability brainstorm
- Need a logo concept brief (for designer handoff)

## Core Deliverables
| Asset | Description |
|-------|-------------|
| Brand Names | 10-15 name options with rationale |
| Taglines | 5-7 options across different positioning |
| Color Palette | Primary + secondary + accent + neutral (hex codes) |
| Typography | Display + body + mono font pairings |
| Logo Concept | Shape, symbol, wordmark direction brief |
| Brand Voice | Tone descriptors + do/don't examples |
| Brand Story | Origin story + mission + vision + values |
| Competitor Differentiation | Brand gap analysis |

## Usage Examples

```
"Create a brand identity for a free online PDF tool for Indian students"
"Generate a color palette for a dark-themed developer tools website"
"Write brand voice guidelines for ISHU TOOLS — friendly, helpful, fast"
"Create 15 brand name ideas for a free image compression website"
"Design a complete brand system for my SaaS startup targeting SMBs"
```

  - "Everything you need. Completely free."
  - "770+ tools. Zero cost. No signup."
  - "The free toolkit every student deserves."
  - "From PDFs to pixels — all free, all instant."

### Recommended Color Palette
```css
--brand-primary: #007AFF;    /* Apple Blue — trust + technology */
--brand-accent: #FF9500;     /* Orange — energy + action */
--brand-success: #34C759;    /* Green — completion + success */
--brand-dark: #000000;       /* True black — premium dark theme */
--brand-surface: #0A0A0C;   /* Near-black — card surfaces */
--brand-text: #F0F0F2;      /* Off-white — legible on dark */
```

### Typography
```
Display Font: SF Pro Display / Clash Display (headings, hero)
Body Font: SF Pro Text / Plus Jakarta Sans (body, labels)
Mono Font: JetBrains Mono (code, technical data)
```

### Brand Voice
✅ **Do**: Be direct and helpful. Use plain language. Celebrate free access.  
❌ **Don't**: Use jargon. Oversell. Make users feel stupid. Use corporate speak.

**Voice Descriptors**: Confident, Helpful, Fast, Student-Friendly, Honest

## Logo Concept Brief
```
ISHU TOOLS Logo Direction:
- Type: Wordmark + Symbol combination
- Symbol: IT monogram in a rounded square (like app icon)
- Color: Blue (#007AFF) primary, white on dark
- Weight: Bold, geometric, modern
- Feel: Professional but approachable
- Size test: Must work at 16px favicon and 300px full logo
```
