# Skill: travel-assistant

## Purpose
Plans complete travel itineraries — day-by-day schedules, accommodation recommendations, transport options, visa requirements, packing lists, budget estimates, local food guides, and safety tips for domestic Indian and international travel.

## When to Use
- User wants a travel itinerary for a specific destination
- User needs visa requirements for a country
- User wants budget breakdown for a trip
- User needs a packing list for a specific trip type
- User wants train/flight options between cities
- User wants hotel recommendations by budget category
- User needs to know local transport options

## Itinerary Structure Template

```
TRAVEL PLAN: [Destination]
Duration: [X days / X nights]
Travelers: [Solo / Couple / Family / Group of X]
Budget: [₹/day per person]
Travel Style: [Backpacker / Budget / Mid-range / Luxury]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## Visa Requirements (Indian Passport)

### Visa-Free Countries (2026 — No Visa Needed)
```
Southeast Asia:
• Thailand: 30 days visa-free
• Indonesia: 30 days visa-free
• Malaysia: 30 days visa-free
• Cambodia: 30 days on arrival
• Vietnam: 45 days visa-free (e-visa available)

Others:
• Mauritius: 90 days visa-free
• Maldives: 30 days on arrival
• Sri Lanka: e-visa (easy, $35)
• Nepal: Free for Indians (open border)
• Bhutan: INR 1200/day tourism fee + permit
```

## Packing List Generator

### Beach Holiday (7 Days, Hot Weather)
```
CLOTHING:
□ T-shirts (5) — light, quick-dry
□ Shorts (3) — swim-worthy
□ Swimwear (2)
□ One casual dress/shirt for evenings
□ Light rain jacket (monsoon season)
□ Flip flops + 1 pair walking shoes

TOILETRIES:
□ Sunscreen SPF 50+ (very important!)
□ After-sun lotion
□ Insect repellent (mosquitoes at beach evenings)
□ Travel-size toiletries

TECH:
□ Portable charger (power bank)
□ Waterproof phone pouch
□ Universal travel adapter

DOCUMENTS:
□ Aadhaar card (Indian travel)
□ Passport (if international)
□ Hotel confirmation printouts
□ Travel insurance documents
□ Emergency contact list (written, not just in phone)
```
