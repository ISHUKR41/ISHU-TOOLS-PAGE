# Skill: competitive-analysis

## Purpose
Conducts structured competitive intelligence — analyzes competitor products, pricing, positioning, strengths, weaknesses, and market gaps — to help build better products and marketing strategies.

## When to Use
- Launching a new product and need to understand the market
- Want to position your product against specific competitors
- Need to identify gaps in the market that aren't being served
- Need a feature comparison matrix
- Want to understand competitor pricing strategies
- Need SWOT analysis of specific competitors

## Core Outputs
| Analysis | Contents |
|----------|----------|
| Competitor Overview | Company facts, funding, team size, product summary |
| Feature Matrix | Side-by-side feature comparison table |
| Pricing Analysis | Pricing tiers, strategy, hidden costs |
| SWOT Analysis | Strengths, Weaknesses, Opportunities, Threats |
| Positioning Map | 2x2 grid mapping competitors on key dimensions |
| SEO Gap Analysis | Keywords competitors rank for that you don't |
| Review Mining | Common complaints from G2/Trustpilot/Reddit |
| Market Gap Report | Underserved segments and missing features |

## Usage Examples

```
"Analyze competitors of ISHU TOOLS: ilovepdf.com, pdfcandy.com, smallpdf.com"
"Create a feature comparison matrix for free PDF tools"
"What are the weaknesses of ILovePDF that ISHU TOOLS can exploit?"
"Find market gaps in the free online tools space for Indian students"
"Compare pricing: Canva vs Adobe Express vs Figma for a design tool"
```

## ISHU TOOLS Competitor Analysis

### Primary Competitors
| Competitor | Strengths | Weaknesses | Opportunity |
|-----------|-----------|------------|-------------|
| ILovePDF | Brand recognition, UI polish | Paid limits, not India-focused | Better free tier |
| Smallpdf | Clean UX, speed | Aggressive paywall (2 free/day) | Unlimited free tools |
| PDF Candy | Many tools, free | Slow processing, poor mobile | Fast mobile experience |
| Convertio | Many formats | Quality issues, paywalls | Better accuracy |
| Canva | Design power | Not a utility tool | Utility + design combo |

### Market Gap Analysis (For ISHU TOOLS)
1. **India-specific tools**: Aadhaar mask, PAN number formatter, Indian exam tools
2. **Student-focused**: GPA calculator, assignment tools, citation generators
3. **Unlimited free**: All tools unlimited, no daily caps
4. **Mobile-first**: Most competitors have poor mobile UX
5. **Hinglish UI**: Localized experience for Indian users
6. **WhatsApp share**: Direct WhatsApp sharing of results

### Positioning Statement Template
```
For [TARGET AUDIENCE]
who [PAIN POINT],
[PRODUCT NAME] is a [CATEGORY]
that [KEY BENEFIT].
Unlike [COMPETITOR],
our product [DIFFERENTIATION].
```

### ISHU TOOLS Positioning Statement
```
For Indian students and professionals
who need powerful document and utility tools without paying,
ISHU TOOLS is a free online toolkit
that provides 770+ tools instantly, no signup required.
Unlike ILovePDF or Smallpdf,
our product has zero daily limits and is optimized for Indian users.
```

## Competitive Intelligence Sources
- **Product reviews**: G2, Capterra, Trustpilot, Product Hunt, Reddit
- **SEO data**: Ahrefs, Semrush (competitor keywords, backlinks)
- **Traffic data**: SimilarWeb, Alexa (estimated traffic)
- **Pricing pages**: Direct competitor websites
- **Job postings**: What they're building (hiring for ML → AI features coming)
- **Social media**: Twitter/LinkedIn for product announcements
