# Skill: ai-secretary

## Purpose
Acts as an intelligent AI executive assistant — drafts emails, schedules meetings, writes agendas, summarizes documents, composes professional communications, manages action items, and handles calendar coordination.

## When to Use
- Need to draft a professional email response
- Need a meeting agenda created from bullet points
- Need to summarize a long document or meeting notes
- Need to create action items from meeting transcripts
- Need to write a formal letter, memo, or internal announcement
- Need to handle scheduling communications (rescheduling, cancellation)

## Core Capabilities
| Task | Output |
|------|--------|
| Email Draft | Professional email with subject + body + signature |
| Meeting Agenda | Structured agenda with time allocations |
| Meeting Summary | Key decisions, action items, owners, deadlines |
| Document Summary | TL;DR + key points + recommendations |
| Professional Letter | Formal letter with proper salutation + structure |
| Internal Memo | Corporate memo format |
| Calendar Invite | Event description text for calendar invites |
| Follow-up Email | Post-meeting follow-up with action items |

## Usage Examples

```
"Draft a professional email declining a vendor proposal politely"
"Create a 1-hour product review meeting agenda for 5 people"
"Summarize these meeting notes into key decisions and action items"
"Write a formal apology email to a client about a delayed delivery"
"Draft an internal announcement about our new remote work policy"
```

## Email Templates by Type

### Professional Reply Template
```
Subject: Re: [Original Subject]

Dear [Name],

Thank you for [reaching out / your email / your proposal].

[Main response - 2-3 sentences]

[Next step or CTA]

Best regards,
[Your Name]
[Title] | [Company]
[Phone] | [Email]
```

### Meeting Request Template
```
Subject: Meeting Request: [Topic] — [Date/Time Suggestion]

Hi [Name],

I'd like to schedule a [duration] meeting to discuss [topic].

Proposed slots (IST):
• [Option 1: Day, Date, Time]
• [Option 2: Day, Date, Time]  
• [Option 3: Day, Date, Time]

Please let me know which works best.

[Video link if applicable]

Best,
[Name]
```

## Meeting Agenda Format
```
MEETING AGENDA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Meeting: [Title]
Date: [Date]
Time: [Start] – [End] (Duration)
Attendees: [Names and Roles]
Facilitator: [Name]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. WELCOME & OBJECTIVES (5 min)
   • Introductions if needed
   • Meeting goal: [stated goal]

2. [AGENDA ITEM 1] (15 min)
   • Owner: [Name]
   • Discussion points:
     - [Point A]
     - [Point B]
   • Decision needed: [Yes/No]

3. [AGENDA ITEM 2] (20 min)
   ...

4. ACTION ITEMS & NEXT STEPS (10 min)
   • Review open items
   • Assign owners and deadlines

5. WRAP-UP (5 min)
   • Next meeting date
   • Any final questions
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Pre-read materials: [Links]
```

## Professional Email Tones
| Tone | When to Use | Key Phrases |
|------|-------------|-------------|
| Formal | Legal, senior stakeholders | "I wish to inform", "Please be advised" |
| Professional | Standard business | "Thank you for", "I'd like to" |
| Warm-Professional | Clients, partners | "Hope you're well", "Looking forward to" |
| Assertive | Difficult situations | "As we discussed", "To clarify" |
| Apologetic | Mistakes, delays | "I sincerely apologize", "I take full responsibility" |
