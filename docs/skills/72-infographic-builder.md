# Skill: infographic-builder

## Purpose
Plans, structures, and generates content for infographics — creating data visualization concepts, information hierarchies, visual storytelling flows, and the underlying content/data that would be rendered as an infographic image or interactive visualization.

## When to Use
- Need to present data or statistics in a visually compelling way
- Need to explain a process, workflow, or comparison visually
- Need educational content for social media (Instagram, LinkedIn)
- Need a timeline of historical events or product roadmap
- Need a comparison between two or more items
- Need a "how-it-works" explainer for a product

## Infographic Types

### 1. Statistical / Data Infographic
Presents numbers and statistics with visual emphasis.
```
Structure:
[Big Number] — [Context sentence]
[Chart/Graph] — [Data series]
[Key Insight] — [What the data means]
Source: [Citation]
```

### 2. Process / How-It-Works
Step-by-step flow showing how something works.
```
TITLE: "How ISHU TOOLS Compresses Your PDF in 5 Steps"
Step 1: 📤 Upload your PDF (drag & drop or click)
Step 2: 🔍 Our engine analyzes the file structure
Step 3: ⚙️ Smart compression applied (removes redundant data)
Step 4: ✅ Quality verified (no visual loss)
Step 5: 📥 Download your compressed file instantly
```

### 3. Comparison Infographic
Side-by-side comparison of two or more items.
```
Feature Matrix:
               ISHU TOOLS    Smallpdf    ILovePDF
Free tools     770+          21          46
Daily limit    Unlimited     2 files     Unlimited
File size      100MB         5MB         25MB
Mobile UX      ⭐⭐⭐⭐⭐    ⭐⭐⭐       ⭐⭐⭐
No signup      ✅            ❌           ✅
```

### 4. Timeline Infographic
Events or milestones in chronological order.
```
ISHU TOOLS Journey:

         └─ Launched with 50 PDF tools
```

## Content Writing Formula

### For Statistical Infographics
```
[Shocking Stat] — [Source, Year]
"[What this means in simple terms]"
[Follow-up comparison or trend]
```
Example:
```
📊 7.87 BILLION files processed since 2016
That's more than the entire world population
↑ Growing 40% year-over-year
Source: PDF Candy, 2026
```
