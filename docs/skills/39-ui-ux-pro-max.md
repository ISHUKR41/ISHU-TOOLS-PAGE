## Key UX Guidelines (99 Total)
1. **Touch Targets**: Minimum 44×44px on mobile
2. **Loading States**: Always show skeleton/spinner for >200ms operations
3. **Error Messages**: Be specific, actionable, human
4. **Empty States**: Explain WHY empty and what to do
5. **Confirmation**: For destructive actions only (not every action)
6. **Progress**: Show progress for operations >2 seconds
7. **Contrast**: WCAG AA minimum (4.5:1 for body text)
8. **Typography**: Maximum 3 type sizes in one viewport
9. **Animation**: Motion should serve function (not decoration)
10. **First Load**: Critical content visible in <1 second
